A simple Notes App built with HTML, CSS, and JavaScript. Users can create, edit and delete notes through a clean popup interface. This project was built as part of my JavaScript learning journey to practice DOM manipulation and event handling.


FEATURES
- Add new notes
- Edit existing notes
- Delete notes
- Popup form for creating and editing notes
- Simple and clean user interface

Build with :
- HTML5
- CSS
- JavaScript (Vanilla JS)
